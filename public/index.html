EnviroPi

/YouTube Description

/YouTube Demo

/GitHub

The EnviroPi is a product that monitors a local environment using a Raspberry Pi B+ and sensors (camera, temp, motion, etc.) to perform logic based outputs based on inputs, take photos, and record results to a database for later analysis. We were able to transmit photos over an HTTP POST request, then using Cloudinary image hosting, display photos on the website taken from the Raspberry Pi Camera module.


Tech used: HTML, CSS, JavaScript, Node.js, AngularJS, Python/Python Flask, SQL, SQLAlchemy, Raspberry Pi, Digital Temp Sensor, Camera Module, Heroku, Firebase, Cloudinary.
